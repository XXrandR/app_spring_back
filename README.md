# Core of ViseApp

## Requirements
- Java 17 min.
- Gradle 7.0 min.

## Quick usage
- To run the project:
```bash
./gradlew bootRun
```
- To build an ALPHA of the project run:
```bash
./gradlew bootJar
```
- The build jar is located in build/libs/:
```bash
java -jar build/libs/"Object.jar"
```
Remember replace "Object.jar" with your own filename.
